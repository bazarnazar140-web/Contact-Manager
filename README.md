# Менеджер контактів

## Опис проєкту

«Менеджер контактів» — консольна програма мовою програмування C++, розроблена в рамках навчальної практики зі спеціальності 121 «Інженерія програмного забезпечення».

Програма дозволяє зберігати та керувати контактами користувачів.

## Основні можливості

* додавання контакту;
* перегляд усіх контактів;
* пошук контакту за ім'ям;
* редагування номера телефону;
* видалення контакту;
* збереження даних у файл;
* завантаження даних при запуску програми;
* обробка некоректного вводу.

## Структура контакту

* Ім'я
* Телефон
* Email

## Формат збереження даних

Контакти зберігаються у файлі `contacts.txt`.
